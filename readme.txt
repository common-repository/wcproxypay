Proxypay payment gateway

== Description == 

Official Proxypay payment gateway.

Main features:

*    Allow you to add a new payment gateway

Please note that you must have been provided some configuration informations
from the Proxypay team to be able to use this plugin.

== Installation == 

This section describes how to install the plugin and get it working.

1. Activate the plugin through the 'Plugins' screen in WordPress
2. Use the Woocommerce->Settings->Payments screen to configure the plugin
